# CF 研究人工复核指南

本文用于 R55/R59 之后的人工审计。它不是交易规则，而是研究解释口径的复核清单。

## 事件阈值复核

需要复核的事件类型：

- 趋势起点：`S1 -> S2`
- 趋势中继：`S2` 延续
- 衰竭观察：`S2 -> S3`
- 终点确认：`S3 -> S4` 或 `S3 -> S0`
- 主力合约切换事件
- 持仓异常变化事件
- 曲线结构突变事件

复核方法：

- 对持仓异常和曲线突变至少比较 `0.90`、`0.95`、`0.975` 三档分位阈值。
- 每档阈值检查事件数、年度分布、方向命中率、1/3/5/10/20 日后验收益均值和中位数。
- 优先使用 R60 `event_threshold_sensitivity` 汇总表和年度分布表作为复核底稿。
- 样本过少、集中在单一年份、成本后失效、或只在单一行情阶段有效的阈值标记为 `WATCH` 或 `REVISE`。
- forward return 只能作为历史后验验证标签，不得进入 latest signal-only 日报。
- 每次复核必须给出 `KEEP` / `WATCH` / `REVISE` / `REJECT`。

最低记录字段：

- reviewer
- review_date
- event_type
- threshold_version
- tested_thresholds
- selected_threshold
- sample_count
- yearly_distribution
- decision
- reason
- next_review_date

## 基本面解释复核

需要复核的字段：

- 数据来源：iFinD、Wind、郑商所、TTEB 等。
- 单位：万吨、吨、张、元/吨、%、天等。
- 频率：日、周、月。
- 统计期：尤其是进口、库存、纺织链周度数据。
- 发布时间：如果无法确认发布日期，不得把统计期直接当成交易日可见信息。
- 口径切换：指标停更、名称更换、来源替换、统计范围调整。

方向解释默认口径：

- 仓单增加：通常解释为潜在交割供应压力上升。
- 仓单下降：通常解释为交割供应压力缓和。
- 库存增加：通常解释为供应/需求压力偏弱。
- 库存下降：通常解释为供应压力缓和或需求改善。
- 进口数量增加：通常解释为供应补充增强，但需复核统计期和到港/清关口径。
- 基差扩大：通常解释为现货相对期货偏强，但必须复核现货地区和期货活跃合约口径。
- 下游开工/负荷改善：通常解释为需求端改善，但必须结合库存和利润验证。

硬性边界：

- 基本面当前只作为解释上下文，`fundamental_signal_status=not_connected`。
- 基本面不进入 `signal_matrix` 或 `composite_score`。
- 周频和月频指标不得自动当成日频信号。
- 无法确认来源、单位、统计期或发布日期时，标记 `HUMAN_REVIEW_REQUIRED`。
- 单一基本面事件不得直接生成交易结论。

## 复核输出

建议每周复核一次事件阈值和基本面解释，每月复核一次参数稳定性和口径切换。

最低输出：

- 复核结论表：CSV 或 Markdown。
- 异常项清单：缺失、停更、口径切换、样本不足。
- 变更说明：如果阈值或方向解释发生调整，说明旧口径、新口径、影响范围。
- 下次复核日期。

## 研究边界

- 本指南只服务研究解释，不构成交易指令。
- 历史收益标签只用于后验验证。
- 任何规则进入正式交易前必须单独通过执行时点、成本、合约切换和风险约束复核。
